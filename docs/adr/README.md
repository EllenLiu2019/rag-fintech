# Architectural Decision Records (ADR)

This directory contains the Architectural Decision Records (ADR) for the project.

## What is an ADR?

An Architectural Decision Record (ADR) is a document that captures an important architectural decision made along with its context and consequences.

## Format

We follow a standard format for ADRs. Please see [TEMPLATE.md](TEMPLATE.md) for the structure.
